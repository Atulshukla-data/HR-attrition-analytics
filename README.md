# HR-attrition-analytics
🔹 Project Title
HR Employee Attrition & Workforce Analytics
🔹 Objective
To analyze employee data and predict attrition using Python, SQL, and Power BI.
🔹 Tools Used
Python (Pandas, NumPy, Scikit-learn)
SQL Server
Power BI
Jupyter Notebook
🔹 Key Steps
Data Cleaning
Exploratory Data Analysis (EDA)
Feature Engineering
Logistic Regression Model
Dashboard Creation in Power BI
🔹 Key Insights
High attrition in low salary group
Overtime employees more likely to leave
Certain departments show higher turnover
